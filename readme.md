# Струтура папок
t9spelling - собственно программа
t9spelling_test - модульные тесты для нее

Для модульных тестов используется gtest, его дистрибутив включен в состав проекта t9spelling_test

# Сборка с помощью scons
Помимо проектов для msvc сделана кроссплатформенная сборка с помощью scons (https://scons.org)

Необходимо установить python и scons (pip install scons), далее в папке с проектом запустить команду "scons".

В данный момент SConstruct-файлы устанавливают флаги в формате msvc, для сборки с помощью mingw или gcc это требуется поправить.
